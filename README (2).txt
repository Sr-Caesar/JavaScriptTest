Form e Tabella

Questo progetto è una semplice applicazione web che permette di compilare un form di iscrizione e visualizzare i dati inseriti in una tabella. È possibile anche effettuare la cancellazione di righe dalla tabella.

Funzionalità:
- Compilazione del form di iscrizione con i campi Nome, Cognome ed Email.
- Validazione dell'indirizzo email inserito.
- Aggiunta dei dati inseriti nella tabella.
- Cancellazione di righe dalla tabella.
- Ricerca di email nella tabella.

Come utilizzare l'applicazione:
1. Aprire il file index.html utilizzando un browser web.
2. Compilare il form di iscrizione inserendo i dati richiesti (Nome, Cognome ed Email).
3. Fare clic sul pulsante "Submit" per aggiungere i dati alla tabella.
4. In caso di errori nella compilazione, verranno visualizzati messaggi di errore sotto i campi corrispondenti.
5. È possibile cancellare una riga dalla tabella facendo clic sul pulsante "Delete" corrispondente alla riga desiderata.
6. Utilizzare il campo di ricerca "Search by Email" per cercare una specifica email nella tabella, poi il match viene scritto a console.

Note:
- Assicurarsi che il file script.js sia correttamente collegato al file index.html.
